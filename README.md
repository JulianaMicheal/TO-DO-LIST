# Todo-List-App History

This is a simple web application to save Your daily tasks in order not to miss anything.

## Features-

---

* Add new task
* Mark task as completed / uncompleted
* Set task priority
* Edit task
* Delete single task
* Delete completed tasks
* Delete all tasks

### Technologies used-

* HTML
* CSS
* JavaScript
